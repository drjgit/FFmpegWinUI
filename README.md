# FFmpegWinUI - WinUI 3 FFmpeg 管理工具

FFmpeg 管理与命令，基于 WinUI 3 开发。提供直观的界面来配置和管理 FFmpeg，让 FFmpeg 的安装和使用变得简单。

## 开发环境

1. **必需组件**

   - Visual Studio 2022
   - Windows App SDK
   - WinUI 3 工具

2. **运行项目**

   首次运行需要右键项目`属性->配置属性->调试->本地计算机`
